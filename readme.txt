== Description ==
This plugin protect your content from being copied by Content Thief. Protect your content without any hassle!!

== Features ==
* Protect your content from selection and copy.
* Admin can disable copy protection for admin users.
* Get full Control on Right click or context menu
* Show alert messages, when user made right click on images, text boxes, links, plain text.. etc
* Disable the keys F12, CTRL+A, CTRL+C, CTRL+X, CTRL+S, CTRL+SHIFT+I or CTRL+V.
* Advanced and easy to use control panel.
* Admin can exclude Home page Or Single posts from being copy protected

== Installation ==
1. Upload the plugin files to the directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the “Plugins” screen in WordPress.
